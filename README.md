# 💻실행 방법

## git clone

```bash
# 폴더 clone 하기
git clone https://github.com/eunjin11/chat-app.git
```

## Frontend

```bash
# 폴더 열기
cd chat-client

# 종속성 설치
npm install

#실행
npm run dev
```

## Backend

```bash
# 폴더 열기
cd chat-server

# 종속성 설치
npm install

#.env 파일에 DB 주소 추가
# MONGODB_URI=MongoDB 주소

#실행
node server.js
```
